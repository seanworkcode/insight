How to ➔ Use an interactive axis

Prerequisites

  • Getting started

Objective

This guide explains how to create a chart with an interactive axis like the one below.

Try using the mouse wheel to zoom in/out of the chart, and click-drag to pan through the data. You can also zoom and pan using touch gestures.

Why use an interactive axis?

Sometimes it can be easier to display a small window of the data, and allow the user to pan the window through the data.

Example

1Prepare the data

For this guide we are generating a data set that consists of a section of a sine wave.

var sinData = []; for (var degrees = 0; degrees < 360 * 3; degrees += 15) { var radians = degrees * Math.PI / 180; sinData.push({ x: degrees, y: Math.sin(radians) + 1 }); } var dataset = new insight.DataSet(sinData);

2Create the chart

Our chart is created with a custom width, height and title.

var chart = new insight.Chart('sin', '#chart') .width(450) .height(250) .title('y = sin(x) + 1');

3Create the axes

We add two linear axes to the chart.

var x = new insight.Axis('x', insight.scales.linear); var y = new insight.Axis('y', insight.scales.linear); chart.xAxis(x); chart.yAxis(y);

4Make the axis interactive

The interactive axis is set using insight.Chart.setInteractiveAxis.

chart.setInteractiveAxis(x);

5Add the line series

Create a line series using the x and y values as key and value functions, then add it to the chart.

var line = new insight.LineSeries('sin-x', dataset, x, y) .keyFunction(function(d){ return d.x; }) .valueFunction(function(d){ return d.y; }); chart.series([line]);

See insight.LineSeries for more detail about what you can do with a line series.

6Draw the chart

chart.draw();
