What is a legend

A Legend shows the title and color of each series in a chart.

How to add a legend

Create a new instance of Legend and add it to the chart. You can also add the title that the legend will display for each series. If we don't provide a title then the legend will display the series name by default.

chart.legend(new insight.Legend()); rows.title('Ages');

Example

Here is the example chart from the Getting Started guide with just the code above added. Notice the Legend in the top right corner with the title of the series and its color.

Further details

Legends are useful when a chart contains multiple series.
