How to change the chart's style

Call the appropriate method on the chart. Then call the chart's draw function to see the change.

Example

Taking the example chart from the Getting started guide, we will change the color of the series. Here is the chart we created in the getting started guide with a default theme.

A series palette specifies the color for each series added to the chart. Since our chart has one series, it uses the first color we specify. Here I've taken the colors from the insight.WarmTheme

chart.seriesPalette(['#A60303', '#FFAD00', '#FF2F00', '#BD7217', '#873300']); chart.draw();

Further details

Every chart is initialized with the default theme which can be overridden. A full list of methods that change the chart's style can be found in the API documentation for chart.
