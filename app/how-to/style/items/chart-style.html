How to change the chart's style

Call the appropriate method on the chart. Then call the chart's draw function to see the change.

Example

Taking the example chart from the Getting started guide, we will change the title's font. Here is the chart we created in the getting started guide with a default theme.

Let's change the title's font and call draw() to see the change.

chart.titleFont('bold 11pt Helvetica'); chart.draw();

It's nice but the change isn't that obvious. Let's change the color of the series. A series palette specifies the color for each series added to the chart. Since our chart has one series, it uses the first color we specify. Here I've taken the colors from the insight.WarmTheme

chart.titleFont('bold 11pt Helvetica'); chart.titleColor('#081717'); chart.seriesPalette(['#A60303', '#FFAD00', '#FF2F00', '#BD7217', '#873300']); chart.draw();

Further details

Every chart is initialized with the default theme which can be overridden. A full list of methods that change the chart's style can be found in the API documentation for chart.
