How to ➔ Customise an Axis

Objective

In this guide we cover how to style an axis:

  1. Show gridlines for an axis
  2. Reverse the position of an axis in a chart
  3. Change the rotation and alignment of tick labels

We also cover how to format the data displayed by an axis:

  1. Change content of tick labels
  2. Setting the maximum and minimum bounds for an axis
  3. Define the tick frequency
  4. Specify an ordering for tick labels

Prerequisites

Before you get started, you need to have read through the Getting Started Guide.

If you want to follow along with the code in the guide, the easiest place to start is with our JSFiddle.

Don't forget: Any changes to a chart are not updated until a draw is called on a chart object or a chart group.

An example : Show gridlines

An axis allows you to add gridlines to the chart using insight.Axis.shouldShowGridlines.

To make it easier to read the values in the chart, we will display gridlines on the horizontal x axis:

x.shouldShowGridlines(true);

To change the position of an axis, the insight.Axis.hasReversedPosition function can be invoked.

We can bring the vertical y axis to the top by calling:

x.hasReversedPosition(true);

The offset and rotation of tick labels is also configurable:

  • insight.Axis.textAnchor — aligns the tick labels to their corresponding 'start', 'middle' or 'end' positions.
  • insight.Axis.tickPadding — controls the spacing between the tick line and tick label (in pixels).
  • insight.Axis.tickLabelRotation — rotates each tick label from horizontal, where angle describes a clockwise rotation (in degrees).
  • insight.Axis.tickLabelOrientation — a helper function that sets the tick label rotation to either 0 or 90.

For our chart, we will reduce the padding for the x axis and rotate the tick labels on the y axis:

x.tickPadding(5); y.tickLabelRotation(30);

An example : Format the data displayed

The content of a tick label can changed by assigning a formatting function to insight.Axis.tickLabelFormat.

To ensure all the names in the chart are upper case, we can include the following code:

y.tickLabelFormat(function(tickValue) { return tickValue.toUpperCase(); });

By default, InsightJS will analyse the data provided in the chart to identify the minimum and maximum bounds. We can override this behaviour by setting our own limits using insight.Axis.axisRange.

To change the maximum value from the default value to 50, we use:

x.axisRange(0, 50);

The tick frequency on the axis automatically calculated by default, so that there is no more than 10 tick labels shown. This can be overridden using insight.Axis.tickFrequency.

To tell the x axis that the tick values should go up by 4, we can call:

x.tickFrequency(4);

For an ordinal axis, it is possible to change the ordering of the tick labels using insight.Axis.orderingFunction. In order for this to have any affect, you also need to call insight.Axis.isOrdered

To sort this data by age, we can include the following code:

y.orderingFunction(function(a, b) { return a.age - b.age; }); y.isOrdered(true);
