Getting Started

Installation

The easiest way to get started with InsightJS is to include the following CDNs in your html page: <link rel="stylesheet" href="//cdnjs.cloudflare.com/ajax/libs/insightjs/1.2.0/insight.min.css"> <script type="text/javascript" src="//cdnjs.cloudflare.com/ajax/libs/crossfilter/1.3.11/crossfilter.min.js"></script> <script type="text/javascript" src="//cdnjs.cloudflare.com/ajax/libs/d3/3.4.11/d3.min.js"></script> <script type="text/javascript" src="//cdnjs.cloudflare.com/ajax/libs/insightjs/1.2.0/insight.min.js"></script> If you would like to install InsightJS manually, the library files are available for download. You will also need to download d3.js and crossfilter, and these should be included in your html before InsightJS.

Objective

We are going to create a basic chart showing the ages of a group of 12 people. By the end of this guide we will have the chart below.

Either read on or take a sneak peek at the code using JSFiddle.

Importing a Data Set

Before using the tools that InsightJS has to offer we will need some data to analyse.

Below we have supplied sample data containing the names, ages and eye colors of 12 different people.

var data = [ { "name": "Michelle Hopper", "age": 26, "eyeColor": "green" }, { "name": "Cochran Mcfadden", "age": 22, "eyeColor": "green" }, { "name": "Jessie Mckinney", "age": 23, "eyeColor": "brown" }, { "name": "Rhoda Reyes", "age": 40, "eyeColor": "brown" }, { "name": "Hawkins Wolf", "age": 26, "eyeColor": "green" }, { "name": "Lynne O'neill", "age": 39, "eyeColor": "green" }, { "name": "Twila Melendez", "age": 26, "eyeColor": "blue" }, { "name": "Courtney Diaz", "age": 20, "eyeColor": "brown" }, { "name": "Burton Beasley", "age": 36, "eyeColor": "green" }, { "name": "Mccoy Gray", "age": 25, "eyeColor": "brown" }, { "name": "Janie Benson", "age": 30, "eyeColor": "green" }, { "name": "Cherie Wilder", "age": 30, "eyeColor": "green" } ];

In order to chart this data, we will first need to create an insight.DataSet object. The data we use to initialise the data set should be an array of objects.

var dataset = new insight.DataSet(data);

Creating a Chart

Now let's walk through the process of creating a simple chart.

In the code below we initialise a chart object with a name and the id of the DOM element where we want to place the chart.

var chart = new insight.Chart('Ages', '#chart') .width(500) .height(350);

We have also set the width and height of this chart in the code above. Without explicitly stating these settings the width and height will default to 300px.

Choosing the Axes

Now, with the chart initialised we need to define the axes. The first decision we need to make is which scale to use for each axis. There are three options:

  • insight.scales.linear
  • insight.scales.ordinal
  • insight.scales.time

In this chart, we are using a linear x-axis for the ages and an ordinal y-axis for the names of people.

We initialise each axis below with a title and the appropriate value from insight.scales.

var x = new insight.Axis('Age', insight.scales.linear); var y = new insight.Axis('', insight.scales.ordinal);

We have provided an Axis Styling, to explain how to carry out more advanced customisations.

With these axes created, we must not forget to associate them with the chart:

chart.xAxis(x); chart.yAxis(y);

Adding a Series

A series object is responsible for transforming data into a visual representation that can be used by a chart.

For this chart, we will use an insight.RowSeries. In the code below we have called this series 'rows' and provided it with our previously created data set and axes.

var rows = new insight.RowSeries('rows', dataset, x, y) .keyFunction(function(person) { return person.name; }) .valueFunction(function(person) { return person.age; });

The x value of a row series is given by insight.RowSeries.valueFunction. In this case we used the age property to get the age of each person in our data set.

Similarly, the y value of a row series is given by insight.RowSeries.keyFunction. In this case we used the name property to get the name of each person in our data set.

Now that we have our row series we need to associate it with the chart object:

chart.series([rows]);

Displaying the Chart

At this point, our chart will still not be visible on the page so we need to tell InsightJS to draw it.

chart.draw();

Congratulations, you have created your first chart!
